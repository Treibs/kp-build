The Rethinking-Memory survey categorizes agent memory into parametric (implicit in weights) and contextual (explicit external, structured/unstructured) forms and defines six core operations: Consolidation, Updating, Indexing, Forgetting, Retrieval, and Condensation.

> This work categorizes memory into parametric (implicit in model weights) and contextual (explicit external data, structured/unstructured) forms, and defines six core operations: Consolidation, Updating, Indexing, Forgetting, Retrieval, and Condensation.
