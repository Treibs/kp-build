MemoryBank implements a forgetting mechanism inspired by the Ebbinghaus Forgetting Curve, allowing the model to forget and reinforce memories based on elapsed time and the relative significance of each memory.

> MemoryBank incorporates a memory updating mechanism, inspired by the Ebbinghaus Forgetting Curve theory, which permits the AI to forget and reinforce memory based on time elapsed and the relative significance of the memory, thereby offering a human-like memory mechanism.

— [[papers/zhong2023memorybank]] (arXiv:2305.10250)
