Augmenting vector RAG with structures like knowledge graphs improves sense-making and associativity but causes performance on basic factual memory tasks to drop considerably below standard RAG.

> Recent RAG approaches augment vector embeddings with various structures like knowledge graphs to address some of these gaps, namely sense-making and associativity. However, their performance on more basic factual memory tasks drops considerably below standard RAG.
