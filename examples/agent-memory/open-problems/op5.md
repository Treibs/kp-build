Episodic memory — single-shot learning of instance-specific, contextually-bound experiences — is argued to be a missing capability distinct from semantic memory and RAG, and no system yet supports all five proposed episodic-memory properties together.

**Why it matters:** Most deployed 'memory' is semantic (facts about the user) or flat retrieval. True episodic memory (explicit reasoning over instance-specific events with contextual binding) is what lets agents learn from a single experience without retraining — the position paper frames this as the catalytic missing piece for long-term agents.
