No current memory agent masters all four core memory competencies (accurate retrieval, test-time learning, long-range understanding, selective forgetting) simultaneously.

**Why it matters:** If a system can retrieve but cannot learn at test time or forget stale facts, it fails in real deployments. The benchmark result shows the field has optimized retrieval in isolation while leaving consolidation, continual learning, and forgetting unsolved — these are the capabilities that distinguish memory from search.
