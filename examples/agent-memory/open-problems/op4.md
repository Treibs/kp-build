It is unresolved whether memory should be parametric (self-updating weights / latent pools) or contextual (external editable stores), and how to combine them; multi-source memory integration is flagged as an open research topic.

**Why it matters:** Parametric memory (MEMORYLLM) avoids retrieval latency and integrates knowledge but is opaque and hard to audit/delete; contextual memory is editable and inspectable but bolted-on. The field lacks a principled account of which knowledge belongs where and how to keep them consistent — directly bearing on auditability, privacy, and forgetting.
