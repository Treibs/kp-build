Does adding structure (knowledge graphs, temporal graphs) to memory help or hurt compared to plain vector retrieval?

### Structure helps multi-hop, associative, and temporal recall
KG/PageRank indexing enables single-step multi-hop retrieval with large accuracy and cost wins; temporal KGs add auditable temporal reasoning and beat MemGPT and baselines.

*Held by:* [[papers/gutierrez2024hipporag]], [[papers/rasmussen2025zep]], [[papers/gutierrez2025hipporag2]]

### Structure can hurt basic factual recall
Graph-augmented RAG improves sense-making and associativity but historically dropped basic factual memory below standard vector RAG — a regression that had to be explicitly fixed.

*Held by:* [[papers/gutierrez2025hipporag2]]
