Is episodic memory a distinct, missing capability, or is semantic memory / retrieval sufficient for long-term agents?

### Episodic memory is the missing piece
Single-shot, instance-specific, contextually-bound episodic memory is qualitatively different from semantic memory and RAG and is the catalytic capability for long-term agents; the field should focus explicitly on its five required properties.

*Held by:* [[papers/pink2025episodic]]

### Reflective/experiential episodic buffers already deliver much of the value
Storing and reflecting on past episodes in an episodic buffer (Reflexion) or memory stream with reflection (Generative Agents) already improves decision-making and believability without weight updates, suggesting practical episodic mechanisms are tractable today.

*Held by:* [[papers/shinn2023reflexion]], [[papers/park2023generative]]

### Both episodic and semantic are facets of a broader memory taxonomy
Episodic and semantic are two of several cognitive mechanisms (alongside sensory, working, procedural); the design question is how to instantiate and coordinate all of them rather than privileging one.

*Held by:* [[papers/huang2026secondhalf]], [[papers/du2025rethinkingmemory]]
