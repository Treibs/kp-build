## Key contributions

- Identifies four core memory-agent competencies: accurate retrieval, test-time learning, long-range understanding, selective forgetting
- Transforms long-context datasets into multi-turn, incremental 'accumulate-then-query' format (incl. EventQA, FactConsolidation)
- Shows current methods (context, RAG, external-memory agents) fall short of mastering all four competencies
