## Key contributions

- Shows a U-shaped positional bias: performance is highest when relevant info is at the start or end of the context and degrades in the middle
- Bias holds even for explicitly long-context models
- Provides the empirical motivation for explicit memory/retrieval over naive long-context stuffing
