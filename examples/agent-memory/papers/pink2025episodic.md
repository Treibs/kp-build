## Key contributions

- Argues episodic memory (single-shot learning of instance-specific contexts) is the missing capability for long-term agents
- Centers five key properties of episodic memory underlying adaptive, context-sensitive behavior
- Outlines a roadmap uniting partial research efforts to support all five properties
