- HippoRAG 2: dual-node (passage + phrase) KG memory with deeper passage integration and online LLM use
- Outperforms standard RAG across factual, sense-making, and associative memory tasks (+7% on associative over SOTA embeddings)
- Frames non-parametric continual learning as a route toward human-like long-term memory
