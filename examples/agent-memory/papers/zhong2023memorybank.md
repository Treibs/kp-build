- Long-term memory mechanism with retrieval, continual updating, and user-personality synthesis across sessions
- Ebbinghaus-Forgetting-Curve-inspired updating that forgets/reinforces memory by elapsed time and significance
- Demonstrated via the SiliconFriend companion chatbot
