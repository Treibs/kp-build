## Key contributions

- Revisits LC-vs-RAG studies after filtering questions answerable without external context
- Finds LC generally outperforms RAG on QA (especially Wikipedia-based), but RAG wins on dialogue/general queries
- Summarization-based retrieval is comparable to LC; chunk-based retrieval lags; stresses context relevance
