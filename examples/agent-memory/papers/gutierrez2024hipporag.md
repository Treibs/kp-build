## Key contributions

- Retrieval framework inspired by the hippocampal indexing theory of human long-term memory
- Orchestrates LLMs, knowledge graphs, and Personalized PageRank to mimic neocortex/hippocampus roles
- Single-step multi-hop retrieval: up to 20% better multi-hop QA, 10-30x cheaper and 6-13x faster than iterative retrieval (IRCoT)
