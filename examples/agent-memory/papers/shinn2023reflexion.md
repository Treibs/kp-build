## Key contributions

- Reinforces language agents via linguistic feedback rather than weight updates
- Stores verbal self-reflections in an episodic memory buffer to improve decisions in subsequent trials
- 91% pass@1 on HumanEval, surpassing the prior GPT-4 state of the art (80%)
