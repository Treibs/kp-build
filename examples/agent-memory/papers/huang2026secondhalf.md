## Key contributions

- Unifies agent memory along three dimensions: substrate (internal/external), cognitive mechanism (episodic/semantic/sensory/working/procedural), and subject (agent-/user-centric)
- Analyzes how memory is instantiated under different agent topologies and learning policies over memory operations
- Reviews evaluation benchmarks/metrics for memory utility and outlines open challenges
