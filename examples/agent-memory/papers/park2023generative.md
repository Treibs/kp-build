## Key contributions

- Memory-stream architecture that stores a complete natural-language record of an agent's experiences
- Reflection mechanism that synthesizes low-level observations into higher-level memories
- Dynamic retrieval (recency + importance + relevance) feeding planning; ablation shows observation, planning, and reflection each contribute to believability
