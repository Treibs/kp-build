## Key contributions

- Memory-centric architecture that dynamically extracts, consolidates, and retrieves salient conversation information
- Graph-memory variant capturing relational structure; consolidation via LLM-driven operations
- On LoCoMo: 91% lower p95 latency and >90% token-cost savings vs. full-context, with accuracy gains (26% over OpenAI on LLM-as-Judge)
