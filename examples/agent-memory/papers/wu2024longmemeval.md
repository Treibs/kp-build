## Key contributions

- 500-question benchmark over five core long-term memory abilities: information extraction, multi-session reasoning, temporal reasoning, knowledge updates, abstention
- Shows ~30% accuracy drop for commercial assistants and long-context LLMs on sustained interactions
- Unified index–retrieve–read framework with optimizations (session decomposition, fact-augmented key expansion, time-aware query expansion)
