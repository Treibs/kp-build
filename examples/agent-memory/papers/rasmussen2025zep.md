## Key contributions

- Graphiti: a temporally-aware knowledge-graph engine synthesizing unstructured conversational and structured business data
- Beats MemGPT on the Deep Memory Retrieval (DMR) benchmark (94.8% vs 93.4%)
- Up to 18.5% accuracy gains with 90% lower latency on LongMemEval-style enterprise temporal reasoning
