## Key contributions

- Categorizes memory into parametric (in weights) vs. contextual (external, structured/unstructured) forms
- Defines six core operations: Consolidation, Updating, Indexing, Forgetting, Retrieval, Condensation
- Maps these to four research topics: long-term, long-context, parametric modification, multi-source memory
