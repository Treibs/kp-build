## Key contributions

- LoCoMo benchmark: very-long-term multi-session dialogues (~300 turns, ~9K tokens, up to 35 sessions) grounded on personas and temporal event graphs
- Machine-human generation pipeline with human verification for long-range consistency
- Shows long-context LLMs and RAG still lag human performance on long-range temporal/causal memory
