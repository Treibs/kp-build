`BarChart.change_bar_values` updates the height of the bars of the chart from a list of values, which does not have to match the number of bars.

> Updates the height of the bars of the chart.

        Parameters
        ----------
        values
            The values that will be used to update the height of the bars.
            Does not have to match the number of bars.
