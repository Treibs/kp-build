Prevalent speculative decoding methods organize predicted tokens as independent chains or fixed token trees, which fails to generalize to diverse query distributions; DySpec instead employs a greedy strategy to dynamically expand the token tree at run time.

> Prevalent methods usually organize predicted tokens as independent chains or fixed token trees, which fails to generalize to diverse query distributions. In this paper, we propose DySpec, a faster speculative decoding algorithm with a novel dynamic token tree structure.
