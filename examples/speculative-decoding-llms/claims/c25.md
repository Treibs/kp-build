SpecInfer verifies the correctness of all candidate token sequences represented by a token tree against the LLM in parallel using a tree-based parallel decoding mechanism, while provably preserving model quality.

> The correctness of all candidate token sequences represented by a token tree is verified against the LLM in parallel using a novel tree-based parallel decoding mechanism. SpecInfer uses an LLM as a token tree verifier instead of an incremental decoder, which significantly reduces the end-to-end latency and computational requirement for serving generative LLMs while provably preserving model quality.

— [[papers/miao2023]] (arXiv:2305.09781)
