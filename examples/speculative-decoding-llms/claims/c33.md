Existing draft model training objectives are not directly aligned with the inference-time goal of maximizing consecutive token acceptance, so PARD-2 reformulates the objective to focus on overall acceptance length rather than token prediction accuracy.

> However, existing draft model training objectives are not directly aligned with the inference-time goal of maximizing consecutive token acceptance. To address this issue, we reformulate the draft model optimization objective, shifting the focus from token prediction accuracy to the overall acceptance length.
