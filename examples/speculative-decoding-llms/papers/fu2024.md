## Key contributions

- Introduces Lookahead decoding, an exact parallel decoding algorithm that accelerates LLM decoding without any draft model or data store
- Trades per-step log(FLOPs) to reduce the total number of decoding steps and is compatible with FlashAttention-style memory-efficient attention
- Achieves up to 1.8x speedup on MT-bench and up to 4x with strong scaling across multiple GPUs on code completion
