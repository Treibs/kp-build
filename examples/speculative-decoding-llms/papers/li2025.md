## Key contributions

- Abandons feature prediction in favor of direct token prediction and uses multi-layer feature fusion via a 'training-time test' technique
- Removes EAGLE's feature-prediction constraint so the draft model benefits fully from scaling up training data
- Achieves up to 6.5x speedup (about 1.4x over EAGLE-2) on chat and reasoning models across five tasks, plus 1.38x throughput in SGLang at batch size 64
