## Key contributions

- Introduces tree-based speculative inference: predictions from small speculative models are organized as a token tree whose nodes each represent a candidate token sequence
- Proposes a novel tree-based parallel decoding mechanism that verifies the correctness of all candidate token sequences against the LLM in parallel
- Uses the LLM as a token tree verifier instead of an incremental decoder, provably preserving model quality while reducing latency
- Reports 1.5-2.8x speedup for distributed LLM inference and 2.6-3.5x for offloading-based inference
