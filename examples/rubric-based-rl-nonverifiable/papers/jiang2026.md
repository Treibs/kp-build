## Key contributions

- Jointly trains a rubric generator and a rubric-conditioned pointwise judge in an alternating GRPO framework whose RL stage uses only pairwise preference data
- Introduces a probability-based scoring rule that reduces evaluation ties caused by hard Boolean rubric aggregation
- Aims to remove dependence on frontier LLMs for rubric judging while retaining competitive reward-modeling accuracy and downstream policy gains
