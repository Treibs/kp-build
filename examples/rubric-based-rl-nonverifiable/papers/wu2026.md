## Key contributions

- Grounds rubric updates in longitudinal training evidence via a persistent evaluation memory (rollout analyses, step-level summaries, rubric update records) rather than local batch/instance views
- Evaluated across science, medicine, instruction following, and creative writing under both global and instance-specific rubric settings, beating static, local-adaptive, and memory-ablated baselines (+2.8 on GPQA-Diamond, +2.2 on IFBench)
- Runs asynchronously alongside the RL loop to reduce blocking latency versus synchronous rubric updates
