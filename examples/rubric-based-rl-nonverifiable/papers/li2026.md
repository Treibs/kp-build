## Key contributions

- Argues rubrics should be the shared interface structuring policy execution, judge feedback, and agent memory for deep-research agents, not merely final-answer evaluators
- Combines stagewise policy decomposition (planning, evidence gathering, review, synthesis conditioned on self-generated rubrics) with Stage-Structured GRPO for denser long-horizon credit assignment
- Trains a shared-backbone reflection meta-policy distilling judged trajectories into reusable rubric-grounded guidance; RubricEM-8B approaches proprietary deep-research systems
