## Key contributions

- Explicitly diagnoses that rubric generation lacks coverage, conflates dimensions, misaligns preference direction, and contains redundant/correlated criteria, degrading judge accuracy and RFT rewards
- Proposes RRD, a recursive decompose-filter framework for rubric refinement with correlation-aware weighting
- Shows refined rubrics yield stronger, more stable RFT reward signals that transfer to held-out benchmarks
