## Key contributions

- Block-wise approximate KV Cache mechanism tailored for bidirectional diffusion models, enabling cache reuse with negligible performance drop
- Confidence-aware parallel decoding that only decodes tokens above a confidence threshold to avoid breaking token dependencies
- Up to 27.6x throughput improvement on LLaDA and Dream with minimal accuracy loss, narrowing the gap with autoregressive models, training-free
