## Key contributions

- Learn2PD: a lightweight adaptive filter that predicts, per token position, whether the current prediction already matches the final output, deciding which tokens can be unmasked in parallel
- Up to 22.58x speedup on the LLaDA benchmark with no performance drop
- Up to 57.51x speedup when combined with KV-Cache
