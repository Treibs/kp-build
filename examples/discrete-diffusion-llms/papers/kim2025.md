## Key contributions

- FlexMDMs: masked diffusion models that generate sequences by inserting and unmasking mask tokens, supporting flexible output length
- Preserves any-order, parallel generation while adding length flexibility
- Fine-tuned LLaDA-8B gains on math (GSM8K 58%->67%) and code infilling (52%->65%)
