## Key contributions

- Pipeline that auto-synthesizes thousands of task instances per repo by breaking existing tests, yielding 50,000 instances from 128 repositories
- An order of magnitude more training data than prior work, with low storage overhead
- Trains SWE-agent-LM-32B to 40.2% Pass@1 on SWE-bench Verified, SOTA among open-source models
