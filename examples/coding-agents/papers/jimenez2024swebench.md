## Key contributions

- Introduces SWE-bench: 2,294 task instances from real GitHub issues and merged PRs across 12 popular Python repositories, evaluated by the repository's own test suite
- Establishes the test-based 'resolve rate' evaluation paradigm the whole agentic-SE line reports against
- Shows even strong LMs with retrieval resolve a tiny fraction (best model Claude 2 at 1.96%), exposing the gap between code completion and real issue resolution
