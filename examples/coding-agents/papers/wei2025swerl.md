## Key contributions

- First approach to scale rule-based RL for real-world SE, rewarding similarity between ground-truth and generated solutions over open-source PR/issue evolution data
- Llama3-SWE-RL-70B reaches 41.0% on SWE-bench Verified, best among medium-sized models
- Shows out-of-domain reasoning transfer (function coding, library use, code reasoning, math, language understanding) despite SE-only training
