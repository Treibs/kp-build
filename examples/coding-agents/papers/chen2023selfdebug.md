## Key contributions

- Introduces Self-Debugging: the model performs 'rubber duck debugging', explaining and inspecting execution results to fix its own code with no human error labels
- Foundational self-correction technique for code generation, with SOTA on Spider (text-to-SQL), TransCoder, and MBPP
- Precursor to the iterative repair/retry loops in modern coding agents
