## Key contributions

- Verbal reinforcement: agents reflect on failure feedback into an episodic memory buffer to improve over trials without weight updates
- Actor/Evaluator/Self-Reflection loop reaches 91% pass@1 on HumanEval (vs 80% for prior GPT-4)
- Seeds the iterative self-debugging/retry behaviour reused by later coding agents
