## Key contributions

- Harder, contamination-resistant benchmark: 1,865 long-horizon enterprise problems across 41 actively maintained repositories
- Three tiers — public (11 repos), held-out (12 repos), and commercial/proprietary (18 repos) — requiring multi-file patches
- Characterizes agent failure modes and provides a testbed that more faithfully captures real-world complexity
