## Key contributions

- Open, sandboxed platform (formerly OpenDevin) for agents that write code, use the command line, and browse the web
- Bundles an agent hub, event-stream architecture, and integrated evaluation across benchmarks including SWE-bench and WebArena
- Became the de facto open-source reference harness for reproducible coding-agent research
