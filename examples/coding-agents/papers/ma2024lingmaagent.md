## Key contributions

- Top-down condensation of a repository into a knowledge graph to reduce complexity for whole-repo understanding (RepoUnderstander prototype)
- Monte Carlo Tree Search (MCTS) based exploration strategy for comprehensive repository understanding before patching
- Reports an 18.5% relative improvement over SWE-agent on SWE-bench Lite
