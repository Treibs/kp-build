## Key contributions

- Decomposes issue resolution into specialized LLM sub-agents, each with a well-defined objective and tuned strategy
- Avoids long inflated trajectories/cost of monolithic agents; achieves 28.33% on SWE-bench Lite
- Evidence point for the plan-and-execute / modular multi-agent decomposition thesis
