## Key contributions

- Introduces the Agent-Computer Interface (ACI): purpose-built file viewer/editor, repo navigation and test-execution commands designed for LM agents rather than humans
- Demonstrates that interface design (not just the base model) drives large gains, reaching 12.5% pass@1 on SWE-bench
- Founds the 'scaffold matters' thesis for coding agents
