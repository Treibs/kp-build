## Key contributions

- Proposes CodeAct: consolidating an agent's action space into executable Python code instead of JSON/text tool calls, enabling control flow and composition
- Reports up to 20% higher success rate vs constrained action formats
- Releases CodeActInstruct tuning data and CodeActAgent; the action-space backbone underlying OpenHands
