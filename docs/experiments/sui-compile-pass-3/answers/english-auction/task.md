Write a module implementing an English auction for a single item (declare a simple `Artifact` object type in the same module). Sellers start an auction as a shared object with a minimum bid in SUI. Bidders bid with `Coin<SUI>`; a higher bid refunds the previous highest bidder immediately. A `settle` function (callable by anyone after the seller closes bidding, or by the seller at any time) sends the item to the winner and the proceeds to the seller; if there were no bids the seller gets the item back.
